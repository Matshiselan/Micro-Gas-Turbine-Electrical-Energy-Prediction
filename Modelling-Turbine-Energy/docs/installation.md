# Installation

## Stable release

To install Modelling Turbine Energy, run this command in your terminal:

```sh
uv add Modelling-Turbine-Energy
```

Or if you prefer to use `pip`:

```sh
pip install Modelling-Turbine-Energy
```

## From source

The source files for Modelling Turbine Energy can be downloaded from the [Github repo](https://github.com/Matshiselan/Modelling-Turbine-Energy).

You can either clone the public repository:

```sh
git clone https://github.com/Matshiselan/Modelling-Turbine-Energy
```

Or download the [tarball](https://github.com/Matshiselan/Modelling-Turbine-Energy/tarball/main):

```sh
curl -OJL https://github.com/Matshiselan/Modelling-Turbine-Energy/tarball/main
```

Once you have a copy of the source, you can install it with:

```sh
cd Modelling-Turbine-Energy
uv sync
```
